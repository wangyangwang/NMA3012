set up faceshift

  1. Install faceshift
  2. Scan all your expressions.
  3. Choose the expression you like, right click and export fit.

prepare your web file

  1. Go to http://threejs.org/examples/#webgl_mirror
  2. Install 3.js library, orbit control and mirror library
  3. copy the code in script tag into a new js file in your own website.
  4. delete sphere group
  5. Go to http://threejs.org/examples/#webgl_loader_obj
  6. load Objloader library into your own web environment.
  7. put your head obj file into model folder of your website.
